# Ant Farm Roadmap

## v0.1 — Ant‑Scout (MVP)
Goal: Help users make safer buying decisions on product pages.

Features:
- Browser extension extracts product data
- Core sends data to Ant‑Scout
- Ant‑Scout returns:
  - summary
  - recommendation
  - confidence
  - risk level

Non‑Goals:
- No automatic purchasing
- No payments
- No multi‑site support

---
